# Credits & Attribution

## Original Work

This project builds upon the great works of:

### installTAK
- **Repository**: https://github.com/myTeckNet/installTAK
- **Author**: JR @ myTeckNet
- **Website**: https://mytecknet.com
- **Purpose**: Universal TAK Server installation automation script

The original installTAK script provides:
- Multi-OS support (Rocky Linux, RHEL, Ubuntu, Debian, Raspberry Pi)
- TAK Server installation wizard
- Certificate generation and management
- Let's Encrypt integration
- Federation Hub support
- Docker container deployment

**Thank you to JR for creating and maintaining this excellent tool that serves the TAK community!**
